逻辑回归

\[ h(\theta) = g(X\theta) \\ g(z) = \frac{1}{1+e^{-z}} \]

g = function(c) {
    1/(1 + exp(1)^-c)
}
x = -10:10
plot(x, g(x) - 0.5, type = "l")


x = -10:10
plot(x, g(x), type = "l")

在线性回归的结果上加上了一个信号函数，将结果规约到（0，1）之间 （注: g 函数的形式选择涉可以通过伯努利分布的最大似然函数推导出来，这部分就略过） 我们怎么计算它呢：类似上面的线性回归 我们先有一个 误差函数（costFuntion 英文原文是这个） \[ J(\theta)=1/(2m)\sum_{i=1}^{m}(h(\theta)_{i}-y_{i})^2 \\ \] 但是 \[ h(\theta) = \frac{1}{1+e^{-X\theta}} \] 我们之前的线性回归函数是一个凸函数

x = seq(-10, 10, by = 0.01)
plot(x, x^2)

加上这个信号函数后，

plot(x, g(x)^2, type = "l")

这是个非凸函数，意味着它木有极值,这时我们就不能用线性回归的方式去计算他的最小值了 必须重新定义误差函数： 逻辑回归的costFuntion 在因为有一个有一个外围的g 函数，为了方便计算我们会在原来的误差函数上加上一个取对数的运算即： \[ J(\theta)=1/m*\sum_{i=1}^{m} \begin{equation} \left\{ \begin{aligned} -log(h(\theta)) :y =1 \\ -log(1-h(\theta) :y =0 \\ \end{aligned} \right. \end{equation} \\ = - 1/m[\sum_{i=1}^{m}y^{i}logh_{\theta}(x^{i})+(1-y^(i))log(1-h_{\theta}(x^{i}))] \] 在加上防止过拟合的惩罚项,完整公式为： \[ J(\theta)= - 1/m[\sum_{i=1}^{m}y^{i}logh_{\theta}(x^{i})+(1-y^(i))log(1-h_{\theta}(x^{i}))] + \lambda/(2m)\sum_{j=1}^{n}\theta_{j}^{2} \] 后面的步骤类似于线性回归了 需要设置的参数，梯度下降的步长\( \alpha \),过拟合惩罚项\( \lambda \),最大迭代次数iterator (注：需要更严谨的数学解释的话，参见概率论的伯努利分布，以及最大似然函数)
